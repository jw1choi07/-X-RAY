## 4. 참고: 업스테이지 멘토링 브리핑 핵심 (원문: 약관X-ray_업스테이지_멘토링_브리핑.pdf)

- **한 줄 요약**: 아무도 안 읽는 약관을 AI가 대신 읽고 위험도 순으로 짚어주는 서비스. ToS;DR 평가 기준 참고 + 한국어 서비스 즉석 분석이 차별점.
- **문제 정의**: 이용자는 자신의 개인정보로 서비스가 무엇을 하는지 모른 채 동의하고, 기존 유일한 대안(ToS;DR)도 한국어 서비스는 커버 못함.
- **실측 근거**:
  - ToS;DR API 직접 호출 → 카카오·쿠팡·토스·무신사 등 9개 검색어 전부 DB 없음
  - 62개 문서 크롤링 성공(50개 시도), 평균 21,000자
  - 파일럿 5개 서비스 + 전체 62개 확장: 원문 근거 확인 64.1%, 매칭 타당성 통과 29.8%
  - Dropbox 사례에서 모델의 "사전지식 오염"(할루시네이션) 실제 포착
- **질문 리스트(멘토링에서 물어볼 것)**: Groundedness Check 모델 폐지 여부, Structured Output vs 프롬프트 강제 JSON, Document Parse 이점, Information Extract 조항 스키마화, 매칭 타당성 낮은 원인, Embed 2 기반 RAG 구조 전환, 크레딧 범위, rate limit, 사전지식 오염 방지 기법
